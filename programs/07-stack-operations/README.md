# Stack Operations

## Problem Statement
Create a class Stack and write a Java program to implement stack operations.

## Language
Java

## Algorithm
1. Read stack size
2. Create stack object
3. Display menu:
   - Push
   - Pop
   - Display
   - Exit
4. Perform selected operation
5. Repeat until exit

## Operations
- Push → Insert element
- Pop → Remove top element
- Display → Show stack contents
