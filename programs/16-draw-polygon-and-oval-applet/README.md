# Draw Polygon and Oval Applet

## Problem Statement
To create a Java Applet program that draws a filled polygon and a filled oval using colors.

## Language
java

## Algorithm
1. Start the program.
2. Import Applet, Graphics, and Color packages.
3. Create a class `NewApplet` extending `Applet`.
4. Override the `paint(Graphics g)` method.
5. Define x and y coordinate arrays for polygon vertices.
6. Set color to Yellow.
7. Draw filled polygon using `fillPolygon()`.
8. Set color to Pink.
9. Draw filled oval using `fillOval()`.
10. Display output.
11. Stop the program.
