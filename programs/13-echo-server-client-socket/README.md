# Echo Server Client Socket

## Problem statement
Write a Java client/server program using Socket Programming (Echo Server).

## language
java

## Algorithm
1. Start the program.
2. Create a ServerSocket on a specified port.
3. Wait for client connection request.
4. Accept the client connection.
5. Create input and output streams for communication.
6. Start the client program.
7. Connect client to server using Socket.
8. Read message from client.
9. Send message to server.
10. Server receives the message.
11. Server sends the same message back (Echo).
12. Client receives echoed message.
13. Repeat communication until user enters "exit".
14. Close client socket.
15. Close server socket.
16. Stop.
