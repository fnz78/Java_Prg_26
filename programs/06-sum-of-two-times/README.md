# Sum of Two Times

## Aim
Write a Java program to create a class `Time` with data members:
- hour
- minute
- second

Create two objects and find the sum of two times using Constructor.

## Algorithm
1. Create class `Time` with hour, minute, second.
2. Use constructor to initialize values.
3. Create method `add()` to add two time objects.
4. Handle carry for seconds and minutes.
5. Display both times and their sum.
