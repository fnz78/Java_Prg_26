# Add Items to List using Java AWT

## Problem statement
To create a Java AWT program that allows the user to enter text in a TextField and add it to a List using a Button click.

## language
java

## Algorithm
1. Start the program.
2. Create a TextField for input.
3. Create a Button labeled "Add".
4. Create a List component.
5. Add all components to the Frame.
6. Register ActionListener for the button.
7. When button is clicked:
   - Read text from TextField.
   - If text is not empty, add it to List.
   - Clear the TextField.
8. Display updated List.
9. Stop.
