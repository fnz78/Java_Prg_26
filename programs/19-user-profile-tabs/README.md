# User Profile Tabs using Java Swing

## Problem statement
To create a Java Swing program that displays user profile information in multiple tabs using `JTabbedPane`, where each tab contains related details such as Personal Info, Contact Info, and Account Info.

## language
java

## Algorithm
1. Start the program.  
2. Create a `JFrame` window.  
3. Create a `JTabbedPane` component.  
4. Create **Personal Info** tab:
   - Add labels and text fields for Full Name, Date of Birth, and Gender.  
5. Create **Contact Info** tab:
   - Add labels and text fields for Email, Phone, and Address.  
6. Create **Account Info** tab:
   - Add labels and text fields for Username and Password.  
7. Add all tabs to the `JTabbedPane`.  
8. Add `JTabbedPane` to the frame.  
9. Set frame size, close operation, and visibility.  
10. Display the GUI window with user profile tabs.  
11. Stop.
