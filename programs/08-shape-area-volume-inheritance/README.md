# Shape Area and Volume using Inheritance

## Problem Statement
Write a Java program to implement inheritance for different 2D and 3D shapes and calculate their area and volume using method overriding.

## Language
Java

## Concepts Used
- Inheritance
- Hierarchical Inheritance
- Method Overriding
- Runtime Polymorphism
- User Input using Scanner

## Algorithm
1. Create a base class `Shape`.
2. Create subclasses `TwoDim` and `ThreeDim`.
3. Derive classes:
   - Circle
   - Square
   - Triangle
   - Sphere
   - Cube
4. Override methods `area()` and `volume()`.
5. Read user choice.
6. Accept dimensions.
7. Display area / volume.
