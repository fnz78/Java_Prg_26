# Odd Even Thread

## Problem Statement
Write a Java program to print odd numbers and even numbers using two separate threads.

## Language
Java

## Algorithm
1. Read limit n from user
2. Create OddThread object
3. Create EvenThread object
4. Start OddThread
5. Start EvenThread
6. OddThread prints odd numbers up to n
7. EvenThread prints even numbers up to n
8. Display result

## Classes Used
- OddThread
- EvenThread
- OddEvenThread
