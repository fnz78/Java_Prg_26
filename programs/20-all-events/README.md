# All Events Example in Java

## Problem statement
To write a Java program that handles all mouse and keyboard events and displays the event name with coordinates.

## Language
java

## Algorithm
1. Start the program.
2. Create a class `AllEventsExample` extending `Frame`.
3. Implement `KeyListener`, `MouseListener`, and `MouseMotionListener`.
4. Register keyboard and mouse event listeners.
5. Capture keyboard events:
   - Key Pressed
   - Key Released
   - Key Typed
6. Capture mouse events:
   - Mouse Clicked
   - Mouse Pressed
   - Mouse Released
   - Mouse Entered
   - Mouse Exited
   - Mouse Dragged
   - Mouse Moved
7. Store mouse coordinates `(x, y)` when events occur.
8. Display the event message on the window using `paint()`.
9. End the program.
