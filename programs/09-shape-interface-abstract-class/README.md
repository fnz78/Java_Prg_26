# Shape Calculator using Interface and Abstract Class

## Problem Statement
Write a Java program to demonstrate interfaces, abstract classes, inheritance, and polymorphism by calculating area and volume of different shapes.

## Language
Java

## Concepts Used
- Interface
- Abstract Class
- Inheritance
- Method Overriding
- Runtime Polymorphism

## Shapes Implemented
### 2D Shapes
- Circle
- Square
- Triangle

### 3D Shapes
- Sphere
- Cube

## Algorithm
1. Display menu of shapes
2. Read user choice
3. Input dimensions
4. Create object
5. Display shape name
6. Calculate area
7. If 3D shape, calculate volume
8. Display result
