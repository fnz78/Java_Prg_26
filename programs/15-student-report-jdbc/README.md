# Student Report JDBC

## problem statement
Write a Java program using JDBC to connect to a MySQL database, retrieve student records, calculate total marks and percentage, and display the report.


## languages
- Java 
- MySQL



## ALGORITHM
1. Start the program.
2. Load MySQL JDBC Driver.
3. Establish connection with MySQL database.
4. Create SQL Statement object.
5. Execute query to fetch student records.
6. Display report heading.
7. Read each record from ResultSet.
8. Calculate total marks.
9. Calculate percentage.
10. Display student details.
11. Close ResultSet.
12. Close Statement.
13. Close database connection.
14. Stop the program.


## DATABASE TABLE
Table: Students

Fields:
- Roll_no
- Name
- sub1
- sub2
- sub3
- sub4
- sub5
