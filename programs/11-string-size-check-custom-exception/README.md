# String Size Check using Custom Exception

## Problem Statement
Write a Java program to check the size of a string.
If the string length is greater than 10, throw a user-defined exception.
Otherwise, display that the string is accepted.

## Language
Java

## Algorithm
1. Create a custom exception class extending Exception.
2. Read a string from the user.
3. Check the length of the string.
4. If length > 10, throw custom exception.
5. Otherwise, print "String accepted".
6. Catch the exception and display the error message.
