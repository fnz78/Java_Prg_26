# Perfect, Abundant and Deficient Number

## Problem Statement
Write a Java program to check whether a given number is Perfect, Abundant, or Deficient.

## Language
Java

## Algorithm
1. Read a number n
2. Find all proper divisors of n
3. Calculate their sum
4. If sum = n → Perfect Number
5. If sum > n → Abundant Number
6. If sum < n → Deficient Number
7. Display result
