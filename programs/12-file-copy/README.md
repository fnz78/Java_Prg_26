# File Copy Program

## Aim
To write a Java program to copy the contents of one file to another file using command line arguments.

## Language
java

## Algorithm
1. Start the program.
2. Read source file name from command line.
3. Read destination file name from command line.
4. Open source file using FileInputStream.
5. Open destination file using FileOutputStream.
6. Read each byte from source file.
7. Write each byte into destination file.
8. Close both files.
9. Display success message.
10. Stop.
