
# Find GCD of Two Numbers

## Problem Statement

Write a program to find the Greatest Common Divisor (GCD) of two numbers.

## Language

Java

## Algorithm

1. Read two numbers `n1` and `n2`
2. Find the larger number and store it in `temp`
3. Initialize `gcd = 0` and `i = 1`
4. Check each number from `1` to `temp`
5. If both `n1` and `n2` are divisible by `i`, store `i` in `gcd`
6. Continue until all values are checked
7. Print the final value of `gcd`
