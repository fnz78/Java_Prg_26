# Student Mark Report Using JDBC

## Problem statement
Create a table student containing Roll No, Name and marks of five subjects.
Write a Java program to read data from the table and display Roll No, Name,
Total Mark and Percentage of each student.

## Languages
- Java 
- MySQL


## ALGORITHM
1. Start program.
2. Load JDBC driver.
3. Establish connection with database.
4. Create SQL statement.
5. Execute SELECT query.
6. Read records one by one.
7. Calculate total marks.
8. Calculate percentage.
9. Display Roll No, Name, Total and Percentage.
10. Close connection.
11. Stop program.
